# 📦 Smart Inventory System

A simple Java console application that manages items, prices, and stock levels using core data structures like Arrays and HashMaps.

## 🚀 About the Project
This is my first application, built to practice fundamental programming concepts such as:
- Data structures (Arrays, HashMaps)
- Object-Oriented Programming (OOP)
- User input handling
- Basic business logic (inventory management)

The application allows users to track and manage inventory items efficiently through a command-line interface.

---

## ✨ Features
- ➕ Add new items to inventory
- 📋 View all items
- ✏️ Update item price or quantity
- ❌ Remove items
- 📦 Track stock levels

---

## 🛠️ Technologies Used
- Java
- Collections Framework (HashMap, ArrayList)
